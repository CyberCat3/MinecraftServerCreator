# MinecraftServerCreator
A Simple tool that can help you make Minecraft servers!

## What is this?
This is a GUI-based tool, that can create Minecraft servers!
It supports Vanilla, Forge, Paper and Fabric.
[Showcase](https://www.youtube.com/watch?v=R67HmD7zm5Q)

## How do I get it?
Download here: <https://jaibel.ddns.net/files/jaibel-server-creator/JaibelServerCreator.jar>

Checksum (SHA256): <https://jaibel.ddns.net/files/jaibel-server-creator/checksums.sha256>

This application is written in Java and requires Java 8+ to run.
